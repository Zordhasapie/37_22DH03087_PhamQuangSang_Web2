Trang web này là gì?

Đây là trang web tổng hợp linh tinh về vài thứ tản mạn xung quanh tựa game League of Legends. Trang web được viết bởi Zord(a.k.a me) với sự trợ giúp của một vài thư viện như Tailwind CSS, JQuery. Trang web này được viết bằng ngôn ngữ HTML, CSS, JavaScript.

Tại sao lại là Tailwind CSS

Tailwind CSS giống như một bản nâng cấp của Bootstrap, với sự tùy biến cao hơn rất nhiều và có nhiều chức năng hơn.
